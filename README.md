# TT Ferry 

A ticketing and reservation system for the Trinidad and Tobago Ferry Service

## Getting Started

First, clone the repo.

Then, while in the root folder (tt-ferry), run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Then open [http://localhost:3000](http://localhost:3000) with your browser to see the application.

To stop the devlopment server, enter Ctrl + C
